# Atoms

## Images

> `Images` as a core element. Has the function as building block for molecules or organisms.

<ul>
  <li>Two ratios are used, 21:9 and 2:3.</li>
  <li>Most images should primarily be displayed in landscape.</li>
  <li>The aspect ratios should be applied to the fullest, with few exceptions.</li>
  <li>The Top Image on pages, also called Hero, can be shown either in three modes: Large, Medium and Small.</li>
  <li>
    In Hero mode the image will use 100% of the web browser width, and ratio needs to be ignored in order to not push
    the page content too far down.
  </li>
  <li>In Medium and Small mode the image should keep the aspect ratio.</li>
  <li>Video (YouTube) should also primarily follow 21:9.</li>
  <li>Upload image size: Recommended 1890x810 pixels. Minimum 1260x540 pixels. Max 2520x1080 pixel.</li>
</ul>{' '}
