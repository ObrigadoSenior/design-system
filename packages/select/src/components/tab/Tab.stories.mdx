# Atoms

## Tab

- Comes both with and without icon.
- Active tab is `red`.
- When there is more tabs than fits the screen, you will be allowed to `scroll/swipe horisontally` to see the rest. This is indicated by a gradient-shadow located to the left/right.
- Max `10 tabs` are allowed on one row.
